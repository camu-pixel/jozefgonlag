Internet of Things

Microcontroller based 
"Spotify current track" wearable display

This is an ESP8266 based Spotify client built with a 256x64 OLED display. The main purpose is to retrieve information about the currently playing track and show it on the display. The ESP8266 connects directly to Spotify when connected to Wifi.
The application uses the Spotify API and uses OAuth 2.0a authorization as described here. Currently-playing endpoint is used to retrieve the track information from the API. Unicode is supported, all characters found in Arial Unicode MS font are embedded in the binary.

Building the software:

The code can be flashed onto the board using esp-open-sdk. After flashing the code onto the device it is configurable by connecting though the Serial port (Baudrate 921600/8-N-1) using PuTTY software. This allows you to manually configure the Spotify API, Wifi SSID and how frequently it should poll the API for new track information. This is set for 10 seconds by default.
